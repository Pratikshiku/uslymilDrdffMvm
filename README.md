## Motor Vehicle License Plate Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/06e3ec8f3a7c4290b02af6bf935a763f/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Vehicle Information
6. Announcement Information
7. License Replacement Application
8. License Application
9. License Transfer Application


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Vehicle Information Management
5. Basic Data Management (Vehicle Type Management, Announcement Type Management)
6. Announcement Information Management
7. License Replacement Application Management
8. License Application Management
9. License Transfer Application Management
10. User Management
